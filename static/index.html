WebDAV 文件访问

通过 WebDAV，您可以访问和管理文件。支持 HTTP 基本认证。只有在同时配置了用户名和密码后，基本认证才会生效。

访问地址: /webdav/

认证用户名: <your_webdav_user> (默认: anonymous)

认证密码: <your_webdav_pass> (默认: <your_sign_key>)
